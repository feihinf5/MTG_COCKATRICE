Note
- Grand Prix-Kyoto 2007 winning deck by Watanabe Yuuya
- Early game stalls for time using [**Remand**](https://api.scryfall.com/cards/581f3780-c480-48c6-b15c-1618f2feccb9?format=image), [**Electrolyze**](https://api.scryfall.com/cards/ef42b5b2-6504-486c-aaa0-9d5e4769ba1d?format=image) and [**Repeal**](https://api.scryfall.com/cards/9e7dd929-4bba-46a6-86c9-b8ed853eb721?format=image)
- Controls the board state using [**Tidings**](https://api.scryfall.com/cards/e28e96c6-4c48-466a-81a0-1137269db529?format=image), fueled by the massive mana generated from Urza's lands paired with the colored mana from [**Izzet Signet**](https://api.scryfall.com/cards/f823be95-bef4-4e86-a924-239be62394bf?format=image)
- Closes out the game using [**Bogardan Hellkite**](https://api.scryfall.com/cards/5852f528-92aa-45cf-8436-5774691676d3?format=image) or [**Demonfire**](https://api.scryfall.com/cards/af2ad333-722e-4d7e-972a-903c24068931?format=image)
- Features a soft-lock combo using [**Teferi, Mage of Zhalfir**](https://api.scryfall.com/cards/3c0145f6-4e27-49e7-9ef6-bac6fa3de26d?format=image) + [**Spell Burst**](https://api.scryfall.com/cards/f95c8015-fd7d-4329-ab23-aec37a824083?format=image) that opponents can only cast spells any time they could cast a sorcery and [**Spell Burst**](https://api.scryfall.com/cards/f95c8015-fd7d-4329-ab23-aec37a824083?format=image) with Buyback to counter opponent spell
- [**Sulfur Elemental**](https://api.scryfall.com/cards/49cba6fd-2474-400d-8568-33fe447abc7c?format=image) strengthening its matchup against Boros Weenie and Control decks
